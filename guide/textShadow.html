文本阴影 ​

text-shadow 属性为文本添加阴影。

基础用法 ​

最简单的用法是只指定水平阴影（2px）和垂直阴影（2px）：

text-shadow: 0px 0px

文字阴影效果！

设置水平阴影
设置垂直阴影
<template>
  <div class="margin-b20">
    <span class="">
      text-shadow: 
      <span class="green bold">{{x}}px {{y}}px</span>
    </span>
  </div>
  <div class="margin-b20">
    <h1 :style="{'text-shadow': `${x}px ${y}px`}">文字阴影效果！</h1>
  </div>
  <el-form-item label="设置水平阴影">
    <el-input-number v-model="x" :min="0"></el-input-number>
  </el-form-item>
  <el-form-item label="设置垂直阴影">
    <el-input-number v-model="y" :min="0"></el-input-number>
  </el-form-item>
</template>

<script setup>
import { ref } from 'vue'
const x = ref(0)
const y = ref(0)
</script>

向阴影添加颜色 ​

text-shadow: 2px 2px red

文字阴影效果！

<template>
  <div class="margin-b20">
    <span class="">
      text-shadow: 
      <span class="green bold">2px 2px {{styleVal}}</span>
    </span>
  </div>
  <div class="margin-b20">
    <h1 :style="{'text-shadow': `2px 2px ${styleVal}`}">文字阴影效果！</h1>
  </div>
  <el-button type="primary" @click="styleVal = 'red'">添加红色阴影</el-button>
  <el-button type="primary" @click="styleVal = 'green'">添加绿色阴影</el-button>
  <el-button type="primary" @click="styleVal = 'blue'">添加蓝色阴影</el-button>
</template>

<script setup>
import { ref } from 'vue'
const styleVal = ref('red')
</script>

向阴影添加模糊效果 ​

text-shadow: 2px 2px 0px red

文字阴影效果！

设置模糊效果
<template>
  <div class="margin-b20">
    <span class="">
      text-shadow: 
      <span class="green bold">2px 2px {{styleVal}}px red</span>
    </span>
  </div>
  <div class="margin-b20">
    <h1 :style="{'text-shadow': `2px 2px ${styleVal}px red`}">文字阴影效果！</h1>
  </div>
  <el-form-item label="设置模糊效果">
    <el-input-number v-model="styleVal" :min="0"></el-input-number>
  </el-form-item>
</template>

<script setup>
import { ref } from 'vue'
const styleVal = ref(0)
</script>
